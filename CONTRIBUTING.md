# Katkıda Bulunma / Contributing

## Türkçe

Bu proje **bireysel kullanım** amacıyla geliştirilmektedir.

Dışarıdan katkı (pull request, fork & merge vb.) kabul **edilmemektedir**. Tüm geliştirme ve bakım işlemleri yalnızca proje sahibi tarafından yapılmaktadır.

Bir hata bulduysanız veya öneriniz varsa, lütfen bir [Issue](../../issues) açın; ancak pull request göndermeyin.

---

## English

This project is developed for **personal use** only.

External contributions (pull requests, fork & merge, etc.) are **not accepted**. All development and maintenance is done solely by the project owner.

If you find a bug or have a suggestion, please open an [Issue](../../issues); however, do not submit pull requests.
