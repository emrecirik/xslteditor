# e-Fatura XSLT Düzenleyici

Bireysel kullanım amacıyla geliştirilmiş, e-Fatura ve e-Arşiv fatura XSLT tasarımları için tarayıcı tabanlı editör.

## 🌐 Canlı Demo

**[https://emrecirik.github.io/xslteditor/](https://emrecirik.github.io/xslteditor/)**

## ✨ Özellikler

- XSLT kod düzenleme (CodeMirror tabanlı sözdizimi vurgulama)
- UBL XML önizleme ve XSLT dönüşümü
- Hazır XSLT ve UBL XML örnekleri (kaynak: [ebelge.gib.gov.tr](https://ebelge.gib.gov.tr/))
- Kendi UBL ve XSLT dosyalarınızı yükleyerek önizleme
- Bölünmüş görünüm (editör + önizleme)
- Sürükle & bırak ile alan ekleme
- Otomatik kaydetme ve kurtarma
- Tablo ve düzen oluşturucu

## 🚀 Kullanım

1. [Canlı demo](https://emrecirik.github.io/xslteditor/) bağlantısını ziyaret edin
2. Hazır örnek XSLT ve XML dosyalarından birini seçin veya kendi dosyalarınızı yükleyin
3. Düzenlemelerinizi yapın ve önizleme ile sonucu kontrol edin
